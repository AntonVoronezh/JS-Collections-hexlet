# Увеличение двумерного массива

 ## Задание
```

Реализуйте функцию enlargeArrayImage, которая принимает изображение в виде 
двумерного массива и увеличивает его в два раза.


Пример:

const arr = [
  ['*', '*', '*', '*'],
  ['*', ' ', ' ', '*'],
  ['*', ' ', ' ', '*'],
  ['*', '*', '*', '*'],
];
// ****
// *  *
// *  *
// ****

enlargeArrayImage(arr);
// ********
// ********
// **    **
// **    **
// **    **
// **    **
// ********
// ********

Также не забудьте внимательно изучить модуль с тестами: здесь расположены 
примеры того, как должна работать функция.




```
